Print Function

Task

Given an integer n, print the numbers from 1 to n on the same line with no spaces or newlines between them. Do not use string methods.

Input

An integer n.

Constraints

1 ≤ n ≤ 150

Example

n = 5

Expected Output

12345

Solution

✨ Complete Solution 
if __name__ == '__main__':
    n = int(input())
    for i in range(1, n + 1):
        print(i, end='')
Hover over code sections to see explanations

Explanation

  • n = int(input()): Reads the number like 5.
  • for i in range(1, n + 1): Loops from 1 to n.
  • print(i, end=''): Prints each number without a newline or space.
  • end='': Stops Python from starting a new line — so all numbers appear together like 12345.
