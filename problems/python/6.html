Print Function

Task

Read an integer n from STDIN.

Without using any string methods, print the following:

123...n

Note that "..." represents the consecutive values in between.

Input

The first line contains an integer n.

Constraints

1 ‚â§ n ‚â§ 150

Example

n = 5

Expected Output

12345

Print the list of integers from 1 through n as a string, without spaces.

Solution

‚ú® Complete Solution 
if __name__ == '__main__':
    n = int(input())
    
    for i in range(1, n+1):
        print(i, end='')

üìñ Understanding the Problem

You need to print numbers from 1 to n, all on one line, without any spaces or line breaks between them.

‚úÖ Correct Output (n=5):

12345

‚ùå Wrong Output:

1 2 3 4 5

üí° How to Solve It:

  1. Take input ‚Äì Read integer n using int(input())
  2. Loop from 1 to n ‚Äì Use range(1, n+1) to iterate
  3. Print without newlines ‚Äì Use print(i, end='')
üîç What Does end='' Do?

By default, print() adds a newline (\n) after each call:

Default behavior:

print(1)
print(2)

Output:

1
2

The end parameter controls what's printed at the end:

  • end='\n' (default) ‚Üí new line
  • end='' ‚Üí nothing (stay on same line)
  • end=' ' ‚Üí space between items

With end='':

for i in range(1, 6):
print(i, end='')

Output: 12345
