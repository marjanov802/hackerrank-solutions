Nested Lists

Task

Given a list of students and their grades:

  • ‚Ä¢ Find the second lowest grade
  • ‚Ä¢ Print name(s) of students with that grade
  • ‚Ä¢ If multiple students, sort names alphabetically

Input

First line: number of students
Then pairs of: name, grade

Example

5 students with grades:

Harry: 37.21
Berry: 37.21
Tina: 37.2
Akriti: 41
Harsh: 39

Expected Output

Berry Harry

Unique scores: [37.2, 37.21, 39, 41]
Second lowest: 37.21

Solution

‚ú® Complete Solution 
if __name__ == '__main__':
    alist = []
    for i in range(int(input())):
        name = input()
        score = float(input())
        alist.append([name, score])
    second_highest = sorted(set([score for name, score in alist]))[1]
    print('\n'.join(sorted([name for name, score in alist if score == second_highest])))
