Division

Task

Read two integers from STDIN, a and b. Add logic to:

  • 1. Print the result of the integer division a // b
  • 2. Print the result of the float division a / b

Input

The first line contains the integer a.

The second line contains the integer b.

Constraints

1 ≤ a, b ≤ 10⁷

Example

a = 3

b = 5

Expected Output

0 0.6

Solution

✨ Complete Solution 
if __name__ == '__main__':
    a = int(input())
    b = int(input())
    
    print(a // b)
    print(a / b)
