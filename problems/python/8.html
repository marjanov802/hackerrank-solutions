Runner-Up Score

Task

Find the second highest score from a list of sports day scores:

  • • Remove all instances of the highest score
  • • Find the maximum of remaining scores
  • • This gives you the runner-up score

Input

Two lines:

  • • n (number of scores)
  • • n space-separated integers (the scores)

Example

Input: 5
2 3 6 6 5

Expected Output

5

Highest: 6 (appears twice)
After removing all 6s: [2, 3, 5]
Runner-up: 5

Solution

✨ Complete Solution 
if __name__ == '__main__':
    n = int(input())
    arr = list(map(int, input().split()))
    max_score = max(arr)
    filtered = [x for x in arr if x != max_score]
    print(max(filtered))

📖 What Is the Question Asking?

You're given a list of scores from a sports day. Your goal is to find the second highest score — also called the runner-up score.

🧠 Example:

Input: 5 2 3 6 6 5

Means:

  • • n = 5 → There are 5 scores
  • • arr = [2, 3, 6, 6, 5]

✔️ Highest score: 6
But there are two of them.
So the next best score after removing all 6s is 5.

💡 Goal: Print 5 — the second highest number in the list.

❗ Why This Doesn't Work (Common Mistake)
n = int(input()) arr = map(int, input().split()) # ❌ Problem here! max_score = max(arr) filtered = [x for x in arr if x != max_score] print(max(filtered))

The issue: map() returns a map object, not a list. It's a one-time-use iterable — once it's looped through (e.g. when you call max(arr)), it's exhausted and becomes empty.

So when you run:

max_score = max(arr) # Uses up the map object filtered = [x for x in arr if x != max_score] # arr is now empty!

✅ Fix: Convert map() to a list:

arr = list(map(int, input().split()))
🔍 List Comprehension Breakdown

Let's fully break down this list comprehension:

[x for x in arr if x != max_score]

This is used to filter out the highest score (so we can find the runner-up).

🧩 Step-by-step breakdown:

Let's say: arr = [2, 3, 6, 6, 5] and max_score = 6

Step x x != max_score Keep it?
1 2 2 != 6 → True ✅ Yes
2 3 3 != 6 → True ✅ Yes
3 6 6 != 6 → False ❌ No
4 6 6 != 6 → False ❌ No
5 5 5 != 6 → True ✅ Yes

✅ Result: [2, 3, 5]

Now you can find the second highest score:

print(max([2, 3, 5])) # → 5
🔢 Complete Example Walkthrough

Given input: 5 and 2 3 6 6 5

Step 1: Read input

n = 5 arr = [2, 3, 6, 6, 5]

Step 2: Find maximum

max_score = max([2, 3, 6, 6, 5]) = 6

Step 3: Filter out all maximum values

filtered = [x for x in [2, 3, 6, 6, 5] if x != 6] = [2, 3, 5]

Step 4: Find runner-up

max(filtered) = max([2, 3, 5]) = 5

✅ Final Answer: 5

💡 Summary
  • • Convert map to list — list(map(int, input().split())) to avoid exhausted iterator
  • • Find maximum — max(arr) gets the highest score
  • • Filter with list comprehension — removes all instances of the maximum
  • • Find runner-up — max(filtered) gives the second highest
